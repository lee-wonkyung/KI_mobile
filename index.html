연금 MBTI TEST — 화면 목록

Figma 디자인을 HTML + CSS(모바일 우선)로 변환한 결과입니다. 색상·여백·폰트 크기는 디자인 수치를 반영했고, 반복 요소(버튼·입력폼·카드)는 css/components.css에 재사용 컴포넌트로 분리했습니다. 이미지·장식 아이콘은 자리표시 박스로 처리했습니다.

파일 구조
· css/tokens.css — 색상·여백·반경·타이포 토큰(커스텀 프로퍼티)
· css/base.css — 리셋·모바일 셸·상태바·홈 인디케이터·유틸리티(u-)
· css/components.css — 버튼·입력폼·카드·헤더·칩·자리표시 박스 등 재사용 컴포넌트
· screens/01–15.html — 각 화면

네이밍 규칙 — 케밥케이스 + BEM(block__element--modifier). 동적 상태는 is- 접두사(is-open·is-filled·is-ready), 유틸리티는 u- 접두사, 이 목록 화면 전용 클래스는 index- 접두사로 네임스페이스합니다.

반응형 — 모바일 우선. 데스크탑(≥600px)에서는 390×844 디바이스 프레임으로 중앙 정렬됩니다.
